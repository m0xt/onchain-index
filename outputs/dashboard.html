Walk-forward methodology Suggest edit ↗

Backtests use BTC daily returns from BMP btc_price, apply the tier allocation to each day's return, and evaluate fixed cycle windows: 2014-2017, 2018-2021, 2022-2024, and 2025-now. Composite inputs are lagged through the rolling z-score helper, so a score dated T uses source data through T-1.
